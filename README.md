Try running some of the following tasks:
- npx hardhat run --network mainnet scripts/deploy.js
- npx hardhat run --network testnet scripts/deploy.js

```shell
npx hardhat accounts
npx hardhat compile
npx hardhat clean
npx hardhat test
npx hardhat node
node scripts/sample-script.js
npx hardhat help
```
